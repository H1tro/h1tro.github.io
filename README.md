# Smart Anti-Smog Air Purification System — Bishkek

## English Description

A production-ready multilingual environmental monitoring dashboard for a Raspberry Pi powered smart anti-smog air purification system in Bishkek. The app uses React, Tailwind CSS, Leaflet, React-Leaflet, OpenStreetMap, leaflet.heat, and Framer Motion to display real Bishkek geography, AQI sensor nodes, PM2.5 heatmap visualization, realtime mock sensor updates, Raspberry Pi system status, filter health, fan telemetry, and responsive light/dark themes.

The public dashboard is read-only, so visitors can inspect air quality data, map markers, popups, charts, and system status without changing sensor or device settings.

## Описание на русском

Production-ready многоязычная панель экологического мониторинга для умной анти-смог системы очистки воздуха на базе Raspberry Pi в Бишкеке. Приложение использует React, Tailwind CSS, Leaflet, React-Leaflet, OpenStreetMap, leaflet.heat и Framer Motion для отображения реальной карты Бишкека, AQI-сенсоров, тепловой карты PM2.5, realtime-симуляции данных, статуса Raspberry Pi, состояния фильтра, телеметрии вентилятора и адаптивных светлой/тёмной тем.

Публичная панель работает только для просмотра: посетители могут смотреть данные качества воздуха, маркеры на карте, popups, графики и статус системы, но не могут менять настройки сенсоров или устройства.
